# Oracle基础知识<br>
## 1、Oracle基本的数据类型<br>
   1、varchar(长度) 字符串<br>
   2、decimal（） 数字<br>
   3、date timestamp 日期<br>
   4、clob blob 特殊文本 二进制<br>
   5、long int integer 整数<br>
   6、char（长度） 字符<br>
   7、number（x，y） x表示总位数 y表示保留小数点后几位数 number（4,2）最大的数是11.11<br>
