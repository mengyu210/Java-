#  二 JBOSS知识点 <br>
1、JBoss支持热部署，也就是war文件部署到服务器上后不需要重新启动JBoss <br>
2、deploy： 这是部署J2EE应用程序（jar、war和ear文件）的位置，只需将相应文件拷贝到该目录下即可<br>
